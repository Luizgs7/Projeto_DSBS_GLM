# Regressão Linear Múltipla: Demanda de energia para aquecer edifícios

Este projeto tem como objetivo, através de uma regressão múltipla, entender a demanda de energia gasta para aquecer prédios (construções de forma geral) como uma função de características dos prédio. O projeto visa aplicar o aprendizado das aulas do programa de Especialização em Data Science e Big Data da UFPR e compor parte da nota na disciplina de Modelos Estatísticos.

Durante todo busquei avaliar e resolver os problemas estatísticas, buscando estimadores consistêntes.

Para acessar o projeto, basta abrir o arquivo [HTML](https://github.com/Luizgs7/Projeto_DSBS_GLM/blob/main/Projeto_Integrador_Luiz_Gabriel_Souza.html) no navegador. É um documento interativo, criado com RMarkdown, onde é possível acompanhar o passo a passo seguido, como em um jupyter notebook.

O projeto é dividido em:

- Análise Exploratória
- Ajuste e seleção de covariáveis
- Análise de diagnósticos
- Interpretação dos componentes do modelo
- Avaliação do ajuste geral do modelo
- Predict do modelo
- Conclusões
- Referências
