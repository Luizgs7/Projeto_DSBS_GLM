# Projeto_DSBS_GLM!

Este projeto tem como objetivo, através de uma regressão múltipla, entender a demanda de energia gasta para aquecer prédios (construções de forma geral) como uma função de características dos prédio. O projeto visa aplicar o aprendizado das aulas do programa de Especialização em Data Science e Big Data da UFPR e compor parte da nota na disciplina de Modelos Estatísticos.

Para acessar o projeto, basta abrir o arquivo HTML no navegador. É um documento interativo, criado com RMarkdown, onde é possível acompanhar o passo a passo seguido, como em um jupyter notebook.
